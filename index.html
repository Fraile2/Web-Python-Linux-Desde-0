Introducción

En esta página aprenderemos a programar en Python desde el cero más absoluto, para ello usaremos el IDE visual studio code en la distribución de GNU/Linux basada en Debian llamada Linux Mint 21.

Quiero destacar que para seguir esta página web, es altamente recomendable usar una distribución de Linux basada en Debian, ya que, en esta página web nos enfocaremos en el uso de comandos en terminal para la instalacion de librerías y paquetes propios de este lenguaje.

Python es un lenguaje de alto nivel, multiplataforma, de código abierto y de propósito general que se ha extendido gracias a la fácil comprensión de su código, puesto a que se parece mucho al inglés.

Es un lenguaje interpretado, eso quiere decir que no necesita de compilación, por eso también se usa como lenguaje de scripts.

También tiene un plus y es que es un lenguaje que hace uso de el paradigma POO lo que lo convierte en un lenguaje en el cual todo es un objeto, cosa que veremos en el tercer trimestre detenidamente.

Instalación de python

Para instalar python en Linux es muy sencillo, para ello, ejecutaremos el siguiente comando:

$ sudo apt install python3 python3-pip

Así es como se descarga python en todas las distribuciones basadas en Debian, a continuación, procederé. a explicar que son estos dos paquetes.

  • python3: Este paquete contiene el intérprete de la última versión de Python versión 3.
  • python3-pip: Este paquete contiene el gestor de paquetes y librerías de la última version de Python3.

Con esto, ya habremos instalado, tanto la última versión estable del intérprete como el gestor de paquetes y librerías de Python3. Ahora veamos como instalar el IDE que vamos a usar para programar en este lenguaje tan polivalente.

Descarga e instalación de Visual Studio Code

Abriremos el primer enlace que os he dejado, y tras ello daremos clic en el botón que pone .deb. De esta forma descargaremos el paquete debian de visual studio code.

Ahora para instalarlo, abriremos una terminal, nos situaremos dónde hemos descargado el paquete deb y ejecutaremos los siguientes dos comandos:

1.- $ mv code_tabulador.deb code.deb

2.- $ sudo dpkg -i code.deb

En el primer comando con tabulador me refiero a que hay que escribir code_ y darle a la tecla tabulador. Esta tecla se situa arriba del Bloq Mayus. Con este comando, renombraremos el paquete debian para que nos sea mucho más fácil y no nos equivoquemos al instalar este paquete.

El segundo comando, simplemente es para instalar el paquete debian. Hay que introducir la contraseña del usuario que estéis usando.

Extensiones necesarias y recomendadas para VSC.

Una vez hayamos instalado visual studio code, hay que instalar las correspondientes extensiones de python.

Abriremos VSC y presionaremos las siguientes tres teclas en órden: Ctrl Shift X y una vez hecho esto se nos abrirá la pestaña de extensiones. Arriba habrá un icono de una lupa y buscaremos e instalaremos primero Python después Pylance, opcionalmente Error Lens adicionalmente, siempre podéis instalar la extensión Rainbow indent para controlar mejor el tema de la identación del código (es decir, cuantas tabulaciones metemos). Dentro de poco, en el PRIMER TRIMESTRE para ser más precisos veremos la importancia que tiene esta extensión a la hora de programar en Python.

Dicho todo esto, puedes darle clic a "PRIMER TRIMESTRE" arriba en la barra de navegación.
