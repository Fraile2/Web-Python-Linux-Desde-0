Introducción

En esta página aprenderemos a programar en Python desde el cero más absoluto, para ello usaremos el IDE visual studio code en la distribución de GNU/Linux basada en Debian llamada Linux Mint 21. Quiero destacar que para seguir esta página web, es altamente recomendable usar una distribución de Linux basada en Debian, ya que, en esta página web nos enfocaremos en el uso de comandos en terminal para la instalacion de librerías y paquetes propios de python.

Instalación de python

Para instalar python en Linux es muy sencillo, para ello, ejecutaremos el siguiente comando:

$ sudo apt install python3 python3-pip
